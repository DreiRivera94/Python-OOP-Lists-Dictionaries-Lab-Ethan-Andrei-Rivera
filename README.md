# Lab 15 – Employee Salary System

## Project Description
An Employee Salary System that allows the user to add employees, display employee records, search for an
employee, and increase an employee’s salary.

## Technologies Used
- Python
- Git
- GitHub
- Visual Studio Code

## Features
This Employee Salary System can:
- Be used to enter multiple employees' name, position, and salary.
- Be used to enter an amount to be added to the initial amount.
- Be used to search a certain employee and their information.

## Classes
The only class used in this project is:
- employeeSystem

### Methods
The methods used in this project are:
- displayEmployees
- searchEmployees
- increaseSalary
- countEmployees

### Attributes
- employee_name
- employee_position
- employee_salary
- amount
